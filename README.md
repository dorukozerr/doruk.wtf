# Doruk.wtf - Personal Website

Welcome to the repository for my personal website, [doruk.wtf](https://doruk.wtf).

```bash

## or git clone https://codeberg.org/doruk/doruk.wtf.git
cd doruk.wtf
yarn
yarn dev
```
